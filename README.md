## Running the code

To replicate the analysis source the `00_master.R` file.

This will run three scripts:
- `01_load_data.R` loads data from online and saves it in  `data/raw`
- `02_clean_data.R` cleans data from `data/raw` and stores cleaned data in `data/processed`
- `03_visualise_data.R` uses plotting functions from `R/plotting_functions.R` to store visuals in `outputs/plots`
